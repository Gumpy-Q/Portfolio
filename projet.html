MALscrap suite

Premier projet sur Python developpé en autonomie après apprentissage du language.
Ce script vise à collecter automatiquement des informations données depuis le site référence de données sur l'animation japonaise, MyAnimeList. A partir d'une saison de départ et de fin, ce script peut récupérer un ensemble d'information sur les séries/films/épisodes spéciaux diffusés pour les extraire dans une sélection de formats exploitables par tous.

Otakulyzer rend les données de l'utilisateur sous format graphique.

  • Python 3
  • Modules BeautifulSoup4, Pandas, Requests, Matplotlib, Seaborn, PySimpleGUI
  • Extract des données en libre accès sur Kaggle
  • Dataviz de l'évolution du marché : nombre d'anime diffusé, part de l'origine des adaptations, évolution du nombre d'épisodes par série, corrélation score/popularité...
